Detailed Description

All vectors in SISL are dense and can be real or complex. Vector operations are defined for real and complex vectors (with function suffix _complex) and functions will fail with an error message if passed an incompatible parameter. The basic data type is sisl_vector_t which should only be accessed using the supplied functions and macros.
